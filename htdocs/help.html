Get help

Post to IGB Help Forum

New in 2020! Post a question to the IGB Help Desk Google Group to get help. Also review IGB Helper posts for tips on using IGB.

File a bug report or request a feature

Use this feedback form to file a bug report or request a feature. This form adds your report or request to our issue tracking site. Also please tell us your email address so we can get in touch.

Email us

You can also email Ann Loraine, who will forward your question to the IGB team. If you don't want your question to be publicly available on our issue tracking site, please let us know.

IGB Developer's Guide

Add new data views, parsers, and visual analytics features as plug-able IGB Apps, which are semi-independent packages users can install and run without having to re-download or even restart IGB.

Visit the IGB Developer's Guide for more information.

Also, feel free to get in touch. We would be happy to help you get started.
