IGB Developer's Guide

Starting with IGB version 6.5, developers can add new functionality to IGB using "plug-ins," semi-independent packages that users can download, install, and run without having to download all-new versions of IGB.

We've also developed a mechanism for users and developers to create live tutorials demonstrating IGB functionality.

To explain how all of this works, we have begun work on an IGB Developer's Guide. Visit the IGB Developer's Guide for more information.
